# Orklys Playground

A workspace for drafting, iterating, and publishing Orklys web pages. Uses GitHub Pages for hosting vetted pages.

## How It Works

All work starts in `projects/`. Each project has its own folder with context docs, copy drafts, and website drafts. When a draft is reviewed and approved, it gets published to `pages/` and goes live on GitHub Pages.

```
projects/  →  iterate & review  →  pages/  →  live on GitHub Pages
```

## Structure

```
projects/           Working drafts and context (not hosted)
├── aarhus-demo/
├── orklys-com/
├── project-page/
└── you-are-screwed-campaign/

pages/              Published pages (hosted via GitHub Pages)
├── orklys-com-v1/
├── orklys-com-v2/
├── project-page-v1/
└── project-page-v2/

reference/          Design system and reference docs
```
